## Keyboard

| Key                | What it does   |
|--------------------|----------------|
| <kbd>F</kbd>       | Fullscreen     |
| <kbd>Space</kbd>   | Pause/Continue |
| <kbd>&#8592;</kbd> | Previous song  |
| <kbd>&#8594;</kbd> | Next song      |
| <kbd>R</kbd>       | Repeat         |
